# 机器人信号板设计方案 (Rev 2.0)

> 基于 **HPM6E80IVM1** 的多总线通信与上位机 USB2.0 汇聚板
> 更新说明: 从 HPM6454IVM2 升级至 HPM6E80IVM1，8路原生CAN-FD (MCAN0-7引脚独立专用)

---

## 1. 项目概述

基于 **HPM6E80IVM1**（双核 RISC-V @ 600MHz）开发机器人信号板，实现机器人控制系统中的多总线接入、边缘通信管理及上位机数据汇聚。

### 核心接口需求

| 接口 | 数量 | 说明 |
|------|------|------|
| 千兆以太网 | 1 路 | 10/100/1000M，RGMII 接口 |
| **CAN-FD** | **8路原生** ⭐ | HPM6E80 自带8路CAN-FD，引脚独立专用零复用 |
| RS485 | 8 路 | 半双工，工业级隔离 |
| UART 串口 | 4 路 | 含 1 路 Debug |
| HLK-7628N 路由器 | 1 路 | 无线/路由扩展模块 |
| USB2.0 | 1 路 | Device 模式，向上位机输出 |

### 数据流

```
传感器/驱动器 → CAN-FD×8, RS485×8, UART×4 → HPM6E80IVM1 → USB2.0 → 上位机
                                                    ↕
                                              HLK-7628N (局域网/无线)
```

## 2. 硬件设计方案

### 2.1 HPM6E80IVM1 最小系统

| 组件 | 规格 | 说明 |
|------|------|------|
| 主晶振 | 24MHz | OSC24M，CL=6pF，ESR 40~80Ω |
| RTC 晶振 | 32.768kHz | CL=9~12.5pF，ESR≤90kΩ |
| 复位 | RSTN (RESET_N) 引脚 | 低电平 ≥300μs |
| **启动配置** | BOOT_MODE = **PA02:PA03** ⭐ | 与旧芯片 PZ07:PZ06 不同！ |
| 外部存储 | SPI NOR Flash (XPI0) | 程序存储，建议 16MB+ |
| **JTAG 调试** | **PA04~PA08** ⭐ | TDO/TDI/TCK/TMS/TRST，非 SWD |

> **重要**: 启动引脚和 JTAG 引脚位置与旧芯片 HPM6454 不同，PCB 布局需调整！

### 2.2 千兆以太网接口 (1路)

- **MAC**: HPM6E80 原生 ENET 控制器 + TSN 交换机
- **PHY**: 外接千兆 PHY（RGMII 接口）
- **接口**: RGMII 连接外部千兆 PHY + RJ45 磁性器件
- **保护**: ESD 防护 + 共模抑制
- **优势**: HPM6E80 自带 TSN 交换机，未来可扩展更多以太网端口

### 2.3 八路 CAN-FD 接口 ⭐ (重大变更)

**关键变更**: HPM6E80IVM1 全 8 路原生 CAN-FD，引脚独立专用无复用：MCAN0→PD00-02, MCAN1→PB04/05, MCAN2→PE08-10, MCAN3-7→PA组。MCP2518FD 已移除。

| 通道 | 方案 | 说明 |
|------|------|------|
| CAN0~CAN7 | **原生** (8路) | HPM6E80 自带 CAN-FD (8Mbps) |


**每路 CAN-FD 通道组成**:

| 组件 | 数量 | 说明 |
|------|------|------|
| CAN 收发器 | 1 | TJA1050T (SOIC-8), 3.3V, CAN-FD 5Mbps+ |
| 共模电感 | 1 | ACM2012-900-2P-T002 (2012封装) |
| TVS 二极管 | 1 对 | PESD1CAN,215 (SOT-23), CANH/CANL→GND |
| 终端电阻 | 1 (可选) | 120Ω ±1% + DIP-4 拨码开关 (4路共享1颗开关) |
| 气体放电管 | 1 | 防雷保护 |
| 数字隔离器 | 1 | ISO7221ADR (SOIC-8), 双通道反向隔离 |
| 隔离电源 | 1 | B0505S-1WR3 或分组共享 (5V→3.3V) |

**HPM6E80 原生 CAN-FD 引脚分配** (289BGA，最终方案):

| CAN 通道 | TXD | RXD | STBY | VIO | 备注 |
|----------|-----|-----|------|-----|------|
| MCAN0 | **PD00(ALT7)** | **PD01(ALT7)** | **PD02(ALT7)** | B06 | ✅ 始终可用(PA00/01仅ISP) |
| MCAN1 | **PB05(ALT7)** | **PB04(ALT7)** | NC(TJA1050T无STBY) | B02 | ✅ 始终可用(PA04/05仅JTAG) |
| MCAN2 | **PE08(ALT7)** | **PE09(ALT7)** | **PE10(ALT7)** | B09 | ✅ 始终可用(PA08仅JTAG) |
| MCAN3 | PA15(ALT7) | PA14(ALT7) | PA13(ALT7) | B01 | ✅ 始终可用 |
| MCAN4 | PA16(ALT7) | PA17(ALT7) | PA18(ALT7) | B01 | ✅ 始终可用 |
| MCAN5 | PA21(ALT7) | PA20(ALT7) | PA19(ALT7) | B01 | ✅ 始终可用 |
| **MCAN6** | **PA24(ALT7)** | **PA25(ALT7)** | **PA26(ALT7)** | B01 | ✅ 始终可用(TSW冲突不用TSN) |
| MCAN7 | PA31(ALT7) | PA30(ALT7) | PA29(ALT7) | B01 | ✅ 始终可用(MDIO已移PF00/PF01) |

| 数字隔离器 | 1 | 隔离 UART 与 RS485 |

### 2.5 四路 UART 串口

| 编号 | 用途 | TX引脚 | RX引脚 | 特性 |
|------|------|--------|--------|------|
| UART0 (开发Debug) | ISP烧录+日志 | PA00 | PA01 | 与CAN0复用，量产释放 |
| UART11 (量产Debug) | 日志输出 | PD15 | PD14 | B07供电，独立于CAN |
| UART3 | 预留业务串口 | 待分配 | 待分配 | 支持 DMA |
| UART6/7 | 预留业务串口 | 待分配 | 待分配 | 支持 DMA |

### 2.6 HLK-7628N 路由器接口

- **UART**: UART1(PA07_TXD/PA06_RXD) 连接 HPM6E80IVM1
- **复位**: PA12(GPIO) → HLK-7628N RST，低电平复位 ≥10ms
- **供电**: 3.3V 独立供电，≥300mA，需 10μF+0.1μF 去耦
- **以太网(可选)**: HLK-7628N 的 ETH_LAN 可接 TSW 交换机端口或独立 PHY
- **启动**: RST 释放后 HLK-7628N 自动启动，BOOT 引脚可选拉高进入恢复模式

### 2.7 USB2.0 上位机接口

- **模式**: USB Device（仅数据上行）
- **保护**: ESD 防护 + 共模器件 + 差分阻抗控制 (90Ω)
- **连接器**: Micro USB 或 Type-C
- **引脚**: USB0_DP / USB0_DN

### 2.8 VCC5V 对外供电接口

- **用途**: 为外部 5V 设备供电（如传感器、执行器）
- **来源**: VCC5V 总线（USB 5V 或 U1 DCDC 输出经肖特基 OR-ing 合并）
- **保护**: 每路 PTC 自恢复保险丝 + TVS
- **连接器**: GH1.25-2P（VCC5V / GND）
- **数量**: 建议 2~4 路，根据实际需求调整

## 3. 电源设计方案

### 3.1 电源树

```
输入源 A: USB 5V ── PTC(500mA) + ESD ──┬── D1(SS34 肖特基)
                                         │
输入源 B: DC Jack 5~12V ── D2(SS34 防反接) ── PTC(3A) ── TVS(SMBJ30A)
              │
              └─ U1 DCDC BUCK (5~12V→5V @3A) ──┬── D3(SS34 肖特基)
                                                  │
                                                  ├── OR-ing 合流点 ── VCC5V 总线
                                                  │
                                                  ├── VCC5V 对外供电连接器 (GH1.25-2P)
                                                  │
                                                  ├── U2 DCDC BUCK (5V→3.3V @3A)
                                                  │       └── VCC3.3V 总线
                                                  │            ├─ HPM6E80 VIO_Bxx/VPMC/VANA/VPLL/VUSB/DCDC_IN
                                                  │            ├─ 千兆以太网 PHY (3.3V)
                                                  │            ├─ CAN×8 收发器 (3.3V)
                                                  │            ├─ RS485×8 收发器 (3.3V)
                                                  │            ├─ (MCP2518FD已移除, 全原生CAN)
                                                  │            └─ HLK-7628N (3.3V, ≥300mA)
                                                  │
                                                  └── U5 LDO (5V→3.0V @50mA)
                                                          └── VBAT (RTC 备份域)
```

### 3.2 HPM6E80 专用电源

| 电源轨 | 电压 | 来源 | 说明 |
|--------|------|------|------|
| **VCC5V** | 5V | USB 5V 直通 / U1 DCDC(5~12V→5V) 经 OR-ing | 对外供电及 VCC3.3V 前级 |
| VCC3.3V | 3.3V | U2 DCDC BUCK (5V→3.3V @3A) | 板级主电源，为 MCU 及外设供电 |
| DCDC_IN | 3.3V | VCC3.3V 总线 | 片上 DCDC 输入 |
| VPMC | 3.3V | VCC3.3V 总线 | 电源管理域 |
| VBAT | 3.0V | U5 LDO (5V→3.0V @50mA) | RTC/备份域 |
| VIO_Bxx | 3.3V | VCC3.3V 总线 | IO 分组供电 |
| VANA | 3.3V | VCC3.3V 总线直连（跳过磁珠，不用 ADC） | 模拟供电 |
| VPLLVUSB | 3.3V | VCC3.3V 总线经 FB 磁珠隔离 | PLL + USB 供电，**磁珠不可省** |
| VDD_SOC | 1.05~1.30V | 片上 DCDC | 核心电压 |
| **VDD_USB** ⚠️ | **~1.0V（内部LDO输出）** | **片上 USB LDO** | **仅接 C3(0.1μF)→GND，严禁接 3.3V！** |
| VDD_PMCCAP | ~1.1V（内部LDO输出） | 片上 LDOPMC | 仅接 C4(4.7μF+0.1μF)→GND |
| VDD_OTPCAP | ~1.1V（内部LDO输出） | 片上 LDOOTP | 仅接 C5(4.7μF+0.1μF)→GND |

> **与旧芯片区别**: HPM6E80 **VDD_USB 是片上 LDO 输出（~1.0V, max 1.3V）而非电源输入**，仅需接 C3(0.1μF)→GND，严禁接 3.3V！

### 3.3 HPM6E80 片上 DCDC 外围元件

HPM6E80 内置 Buck DCDC 转换器，将 VCC3.3V（DCDC_IN）转换为核心电压 VDD_SOC（软件可调，600MHz 时约 1.275V）。

| 位号 | 参考值 | 封装 | 说明 |
|------|--------|------|------|
| **L1** | **4.7μH** (2.2~10μH 可用) | CDRH5D28 / SWPA5040S | DCDC 开关电感，**≥2A 饱和电流**，低 DCR。一端接 **DCDC_LP(R3,T3,U3 短接)**，另一端接 **VDD_SOC 网络(11个焊球全部短接)** |
| **C1** | **47μF**/10V (33~66μF 可用) | 1210 / 1206 | DCDC_IN 输入电容，紧靠 R1,T1 放置，低 ESR |
| **C2** | **0.1μF**/10V X7R × **11 颗** | 0402 | VDD_SOC 去耦，**11颗VDD_SOC焊球全部短接为一个网络，每颗焊球各放1颗C2→GND** |
| **C3** | 0.1μF/10V X7R | 0402 | VDD_USB(M12) 及 DCDC 内部节点去耦 |
| **C4** | **4.7μF** + 0.1μF | 0603 + 0402 | VDD_PMCCAP(M9) 去耦，仅接电容 |
| **C5** | **4.7μF** + 0.1μF | 0603 + 0402 | VDD_OTPCAP(M8) 去耦，仅接电容 |

**VDD_SOC 电压与主频对应关系**:

| 主频 | VDD_SOC 典型值 | 模式 |
|------|---------------|------|
| 600MHz | 1.275V | 性能模式 |
| 480MHz | 1.175V | 平衡模式 |
| 400MHz | 1.075V | 节能模式 |

> **注意**: VDD_SOC 电压由软件通过 DCDC 输出电压寄存器配置，设计时按 600MHz/1.275V 预留散热。
> **VDD_USB(M12) 严禁接 3.3V** — 它也是片上 LDO 输出（~1.0V），仅需 0.1μF 去耦到 GND。

## 4. EMC 与可靠性设计

### 4.1 布局布线原则

1. **高速信号优先**: USB2.0 差分对 (90Ω)、RGMII 走线 (50Ω) 最短等长
2. **分区布局**: 强干扰接口 (CAN/RS485) 与主控核心区分区
3. **地层规划**: 数字地与接口保护地单点连接
4. **通道一致性**: 8 路 CAN-FD 与 8 路 RS485 一致通道布局
5. **晶振布局**: 24MHz 靠近主控，走线短且包地
6. **调试便利**: JTAG (PA04~PA08)、复位按键、LED 布局于板边

### 4.2 保护措施

| 接口 | 保护方案 |
|------|----------|
| CAN-FD | TVS + 共模电感 + 气体放电管 + 可选隔离 |
| RS485 | TVS + 总线偏置 + 可选隔离 |
| 以太网 | 集成磁性隔离 + ESD |
| USB | ESD 防护二极管 |
| 电源入口 | 反接保护 + 过压保护 + 保险丝 + TVS |

### 4.3 可靠性机制

- 独立硬件看门狗
- 电源监控
- 各总线链路超时检测与自动恢复
- USB 断线重连
- 软件异常统计与复位

## 7. 开发与调试接口

- **JTAG** (PA04~PA08): 程序下载和调试（**非 SWD**，量产释放给MCAN1/2）
- Debug UART: 开发=UART0(PA00/PA01)，量产=UART11(PD14/PD15)
- ISP烧录: 开发=UART0，量产=USB0(M16/M17专用引脚)
- 状态指示灯: 电源、运行、错误
- 复位按键: 全局复位

> **注意**: RISC-V 芯片使用 JTAG（非 SWD），调试器需支持 RISC-V JTAG（如 JLink、DAPLink 等）

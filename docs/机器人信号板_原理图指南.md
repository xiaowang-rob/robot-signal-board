> **VPLLVUSB 供电域说明**: 
> - **VPLL (G12)** 和 **VUSB** 在 HPM6E80 上是 **同一个焊球**，数据手册标注为 `VPLL/VUSB`
> - 该焊球同时为 PLL（内部 PLL 时钟电路）和 USB PHY 的模拟部分供电
> - **必须保留磁珠 FB 从 VCC3.3V 隔离**，避免数字噪声导致 PLL 抖动影响 USB/以太网
> - VANA(M11) 不用 ADC 时可以 **VCC3.3V 直连（跳过磁珠）**，仅加去耦电容
> - **USB0_VBUS (H12) 容易混淆**：它不是电源轨，只是 USB 连接器 VBUS 的检测输入引脚（5V 耐受），用于检测 USB 线是否插入

### 2.2 片上 DCDC 电路详解

HPM6E80 内置一个高效 Buck（降压）DCDC 转换器，将 VCC3.3V 转换为核心电压 VDD_SOC（软件可调），**无需外部 DCDC 控制器芯片**。

#### 2.2.1 DCDC 架构

```
VCC3.3V ──┬── C1(47μF/10V 低ESR) ──── GND
          │
          └── DCDC_IN(2颗焊球: R1, T1)
               │
               ┌── [片上DCDC: 高频开关 Buck 转换器]
               │   - 内部高边/低边功率 FET
               │   - 软启动 + 过流保护(4A) + 过压保护(1.6V)
               │   - 多模式 PFM/PWM 自动切换
               │
               DCDC_LP(3颗焊球: R3, T3, U3)  ← 全部短接 → 开关节点(LX)
               │
               L1(4.7μH, ≥2A 饱和电流)
               │
               └── VDD_SOC(11颗焊球: F6,G7,L7,H8,K8,J9,H10,K10,G11,L11,F12)
                    │               ↑
                    │       全部11颗短接成一个VDD_SOC网络
                    │
               C2(0.1μF/10V X7R) × 每引脚（11颗）→ GND，各自就近放置
```

#### 2.2.2 各引脚说明

| 引脚组 | 焊球 | 数量 | 作用 |
|--------|------|------|------|
| **DCDC_IN** | **R1, T1** | **2** | Buck 输入，全部接 VCC3.3V，C1(47μF) 紧靠放置 |
| **DCDC_LP** | **R3, T3, U3** | **3** | Buck 开关节点（LX），全部短接后接 L1 一端，**L1 另一端接 VDD_SOC 网络** |
| **DCDC_GND** | **R2, T2, U2** | **3** | DCDC 专用地，**必须通过独立过孔直达 GND 层**，不与 VSS 混用 |
| **DCDC_SNS** | M6 | 1 | 电流检测，**悬空 NC** |
| **VDD_SOC** | F6,G7,L7,H8,K8,J9,H10,K10,G11,L11,F12 | 11 | **11颗全部短接成同一VDD_SOC网络**，每颗焊球放 C2(0.1μF)→GND |

|------|--------|--------|--------|------|
| 输入电压 (DCDC_IN) | 3.0 | 3.3 | 3.6 | V |
| 输出电压 (VDD_SOC) | 0.6 | - | 1.375 | V |
| 输出电压精度 (Run 模式) | -3% | - | +3% | |
| 输出电压精度 (LP 模式) | -6% | - | +6% | |
| 过流保护阈值 | - | 4 | - | A |
| 过压保护阈值 | - | 1.6 | - | V |

| 档位 | 主频 | VDD_SOC 典型值 | 场景 |
|------|------|---------------|------|
| 性能 | 600MHz | **1.275V** | 全速运行 |

#### 2.2.5 外围元件选型

| 位号 | 值 | 封装 | 说明 |
|------|-----|------|------|
| **L1** | **4.7μH** (2.2~10μH 可用) | CDRH5D28 / SWPA5040S | DCDC 开关电感，**≥2A 饱和电流**，低 DCR。一端接 **DCDC_LP(R3,T3,U3 三颗短接)**，另一端接 **VDD_SOC 网络(11颗短接)** |
| **C1** | **47μF**/10V (33~66μF 可用) | 1210 / 1206 | DCDC_IN 输入电容，低 ESR 电解或 MLCC 多颗并联 |
| **C2** | **0.1μF**/10V X7R × **11 颗** | 0402 | VDD_SOC 输出去耦，**每引脚 1 颗，就近放置** |
| **C3** | 0.1μF/10V X7R | 0402 | DCDC 内部节点去耦（见数据手册表 10 参考电路） |
| **C4** | **4.7μF** + **0.1μF** | 0603 + 0402 | VDD_PMCCAP(M9) 去耦 |
| **C5** | **4.7μF** + **0.1μF** | 0603 + 0402 | VDD_OTPCAP(M8) 去耦 |

#### 2.2.6 布局布线要点

1. **DCDC_IN 走线**: 从 VCC3.3V 经 C1(47μF) 到 R1, T1 两颗 DCDC_IN 焊球，走线宽度 ≥20mil
2. **L1 位置**: 紧靠 DCDC_LP 焊球（R3, T3, U3），所有 3 颗 DCDC_LP 先短接再接到 L1 一端
3. **VDD_SOC 网络**: L1 另一端接 VDD_SOC 网络，**11 颗 VDD_SOC 焊球全部短接成一个 VDD_SOC 网络**（通过 PCB 内层或表层铜皮）。**11 颗 C2(0.1μF) 必须每颗紧贴对应 VDD_SOC 焊球放置**，过孔直落 GND 层
4. **DCDC_GND 独立**: R2, T2, U2 三颗 DCDC 专用地焊球通过独立过孔接 GND 层，**不与 VSS 共用同一过孔**
5. **开关节点**: DCDC_LP(R3,T3,U3)→L1 之间的走线尽量短而宽，下方禁止走敏感信号
6. **反馈路径**: VDD_SOC 去耦电容的地端必须短接到 DCDC_GND，以保证反馈环路

**24MHz 主晶振 (OSC24M)**:

| HPM6E80 引脚 | HPM6E80 功能 | 晶振 (Y1) 引脚 | 说明 |
|-------------|-------------|---------------|------|
| L17 | **XTALI** | **Pin1 (OSC_IN)** | 晶振输入，C1(12~22pF)→GND |
| L16 | **XTALO** | **Pin3 (OSC_OUT)** | 晶振输出，C2(12~22pF)→GND |
| - | - | **Pin2, Pin4** | 接 GND |

```
     HPM6E80              24MHz 晶振
    ┌──────┐             ┌──────────┐
    │      │             │          │
    │  XTAL_IN──┬── C1───┤ Pin1     │
    │          │         │          │
    │          ├── Rf ───┤ Pin3     │
    │          │         │          │
    │  XTAL_OUT─┬── C2───┤          │

    └──────┘             └──────────┘

- C1/C2 值根据晶振负载电容和 PCB 寄生电容调整，公式：CL = (C1×C2)/(C1+C2) + Cstray

**32.768kHz RTC 晶振**:
- 同 24MHz 连接方式
- Y2: 32.768kHz, CL=9~12.5pF, ESR≤90kΩ
- RTC_XTAL_IN/OUT 引脚

### 2.3 复位电路

```
       3.3V
        │
       R1(10kΩ) ──┬── RSTN (RSTN 引脚)
                    │
                   C1(0.1μF)
                    │
                   GND

       可选: 按键复位 ──┬── RSTN
                        │
                       R2(100Ω) (限流)
```

- RSTN 低电平持续时间 ≥300μs
- 建议加入外部电压监控 IC（如 TPS3808），实现电源上电复位

    │ XPI0_D0  ──────────────── SI(D0)      │
    │ XPI0_D1  ──────────────── SO(D1)      │
    │ XPI0_D2  ──────────────── WP#(D2)     │
    │ XPI0_D3  ──────────────── HOLD#(D3)   │
    │       │                    │           │

                                  │      │
                                 VCC    GND

|------|----------------------|-------------|------|
| XPI0_CA_CS0 | **PB30** (A5) | **CS# (Pin1)** | 片选 0 |
| XPI0_CA_SCLK | **PB29** (B4) | **CLK (Pin6)** | 时钟 |
| XPI0_CA_D_0 | **PB27** (B3) | **SI / IO0 (Pin5)** | 数据 0 (MOSI) |
| XPI0_CA_D_1 | **PB28** (A4) | **SO / IO1 (Pin2)** | 数据 1 (MISO) |
| XPI0_CA_D_2 | **PB26** (A3) | **WP# / IO2 (Pin3)** | 数据 2 (写保护) |
| XPI0_CA_D_3 | **PB31** (A5) | **HOLD# / IO3 (Pin7)** | 数据 3 (保持) |

| XPI0_CA_CS1 | **PB24** (A2) | **CS# (第二颗)** | 片选 1 (可选第二颗 Flash) |

- 串行 R1/R2(10~33Ω) 匹配电阻（靠近 HPM6E80 放置）
- 建议选型: Winbond W25Q128JV / W25Q256JV 或 Micron MT25QL256

> **重要**: XPI0 引脚在 **PB24~PB31** (B04 供电分组)，不在 PF 组！PB24~PB31 的 MCAN6/MCAN7 ALT 功能被 XPI0 占用。本设计 MCAN0/1/2/4/5/6/7 已集中到 PD 组(B06/B07)，MCAN3 保留 PA 组。PB24~PB31 的 MCAN 功能被 XPI0 Flash 占用，但 MCAN6/7 已分別移至 PD24/25 和 PD30/31，CAN 全 8 路可用。

## 4. 千兆以太网

### 4.1 RGMII 接口 (HPM6E80 → 外部 PHY)

```
    HPM6E80                   YT8531C 千兆 PHY
    ┌──────┐                    ┌──────────┐
    │      │  RGMII_TXCK ────── TX_CLK     │
    │      │  RGMII_TX_CTL ──── TX_EN      │
    │      │  RGMII_TXD[3:0] ── TXD[3:0]   │
    │      │  RGMII_RXCK ────── RX_CLK     │

#### 工作模式配置（上电锁存）

这些引脚上电时作为配置输入锁存，之后切换为正常功能。**仅需在需要改变默认值时加外部上下拉**。

| YT8531C 引脚 | 锁存功能 | 默认值 | 外部配置 | 说明 |
|-------------|---------|-------|---------|------|
| 32 | **CFG_EXT** | 内部上拉=1 | **上拉 10kΩ→3.3V**（与内部一致） | =1: I/O 电源由**外部 VCC3.3V 直接供电** |
| 33 | **CFG_LDO0** | 内部上拉=1 | **下拉 10kΩ→GND**（需外部下拉） | CFG_LDO[1:0]=**00** → 3.3V RGMII |
| 34 | **CFG_LDO1** | 内部下拉=0 | NC（与内部一致） | |
| 22 | **PHYAD0** | 内部下拉=0 | NC（默认地址 0x00） | |
| 27 | **PHYAD1** | 内部下拉=0 | NC | PHY 地址 0x00 |
| 26 | **PHYAD2** | 内部下拉=0 | NC | |
| 24 | **TXDLY** | 内部下拉=0 | NC（TXC 无延迟，PCB 等长保证） | |
| 25 | **RXDLY** | 内部上拉=1 | NC（RX_CLK 自动加 2ns 延迟） | |
| 23 | **PLLOFF** | 内部下拉=0 | NC（PLL 正常工作） | |
| 35 | **CLKOUT** | - | **NC（浮空）** | 不用 |

> **总结**: 只有 **CFG_LDO0(Pin33)** 需要额外加 **下拉 10kΩ→GND**，其余所有配置引脚默认值已符合 3.3V RGMII 的要求，直接 NC 即可。

> **配置总结（1000M RGMII @ 3.3V）**:
> - **CFG_EXT=1** + **CFG_LDO[1:0]=00** → 外部 3.3V 供电，RGMII I/O 电压 = **3.3V**
> - **PHYAD[2:0]=000** → PHY 地址 **0x00**（MDIO 通信用）
> - **TXDLY=0, RXDLY=1** → TXC 无延迟（PCB 等长走线保证），RX_CLK 加 2ns 延迟
> - HPM6E80 侧 **VIO_B10 接 VCC3.3V**（3.3V RGMII 模式）

#### 隔离侧供电 (isol_3.3V)

每条 CAN 通道的隔离侧(TJA1050T + ISO7221侧2)需要 **isol_3.3V** 供电。

| 方案 | 说明 | 适用 |
|------|------|------|
| A: 各通道独立 DC-DC | 每路用 B0505S-1WR3 (5V→3.3V 1W) | 通道间全隔离 |
| B: 各通道独立 LDO | 从 VCC5V→LDO(3.3V), 需保证 VCC5V 干净 | 隔离栅共享 |
| C: 分组共享隔离电源 | 4路共用1个 DC-DC, 分组隔离 | 平衡成本与隔离 |

> **推荐方案 C**: 8路 CAN 分为 2 组，每组 4 路共享一个隔离 DC-DC (5V→3.3V, 3W)。这样通道组间隔离，组内共地。若要求全隔离，选方案A。

| 参数 | 值 |
|------|----|

| 供电 | 3.3V ±0.2V, 空载170mA, **需≥800mA供电能力(推荐1A+)** |
| WiFi | 2.4GHz 802.11b/g/n, 300Mbps, AP/STA模式 |
| 以太网 | **5口百兆交换机**(PORT0~4), 内置PHY, 需外接磁性RJ45 |

| GPIO | 8+个, 3.3V, 8mA |

**设计要点**:
- **供电**: 3.3V ≥800mA，Pin2和Pin3都要接3.3V，10~22μF大容量+100nF去耦就近放置
- **复位**: Pin14(PORST_N) 低有效，**不要外部上拉/下拉**，用10kΩ+100nF RC延时电路
- **天线**: Pin22(ANT) 需50Ω阻抗匹配走线，接IPEX座或PCB天线，**天线质量直接影响WiFi性能**
- **Boot引脚**: Pin5/27/28/29(SPI Flash总线) **不要加外部上下拉电阻**，悬空即可
- **以太网**: 5口百兆交换机，每口需100Base-T磁性变压器+RJ45，PORT0通常做WAN
- **UART**: 默认固件支持UART0+UART1透明透传，连接HPM6E80的UART3(PC15/PC14)
- **LED**: Pin9~13为以太网LED(P0~P4)，低电平点亮

| ~~**FB_ANA**~~ | ~~600Ω@100MHz, 0805~~ | ~~0805~~ | ~~3.3V → VANA~~ | **本设计不用 ADC，VANA 跳过磁珠，此件不贴** |
| **FB_PLL** | 600Ω@100MHz, 0805 | 0805 | VCC3.3V → VPLL/VUSB(G12) | PLL 磁珠，必须保留 |

### 9.4 DCDC 外围电路参考

**U2 (5V → 3.3V, 3A BUCK)** 参考:
```
    5V ──┬── CIN(22μF) ── VIN ──── SW ── L(2.2μH) ──┬── 3.3V
         │                      │                      │
        GND          BST ── C(0.1μF) ── SW    COUT(22μF) │
                              │                      │
                             FB ── R1(10k) ── 3.3V  GND
                                  │
                                 R2(3.32k)
                                  │
                                 GND
         EN ── 3.3V (使能)
```

> **注意**: HPM6E80 内部 DCDC 只需外部电感 L1 + 电容 C1~C5，无需外部 DCDC 控制器。U2(5V→3.3V) 是为 IO/外设供电的板级电源。

# 机器人信号板设计方案核对清单

> 基于 **HPM6E80IVM1** 的多总线通信与上位机 USB2.0 汇聚板
> Rev 2.0 — 更新至 HPM6E80IVM1（原生 8 路 CAN-FD）

---

## 1. 硬件设计检查

### 1.1 HPM6E80IVM1 最小系统

- [ ] 主晶振 24MHz 选型正确（CL=6pF，ESR 40~80Ω）
- [ ] RTC 晶振 32.768kHz（CL=9~12.5pF，ESR≤90kΩ）
- [ ] 复位电路（RSTN 低电平 ≥300μs）
- [ ] **启动配置引脚 PA02:PA03**（非旧芯片的 PZ07:PZ06！）
- [ ] SPI NOR Flash 选型（XPI0 接口，容量≥16MB）
- [ ] **JTAG 接口 PA04~PA08**（TDO/TDI/TCK/TMS/TRST）
- [ ] **确认使用 JTAG 而非 SWD**（RISC-V 芯片不支持 SWD）
- [ ] 电源去耦电容充足
- [ ] DCDC 电感值正确（2.2~4.7μH）
- [ ] DCDC 输入电容 33~66μF

### 1.2 电源系统

- [ ] 外部输入反接保护
- [ ] 外部输入过压保护
- [ ] 外部输入保险丝
- [ ] 5V DCDC 选型（电流≥3A）
- [ ] 3.3V DCDC 选型（电流≥2A）
- [ ] 1.8V LDO 选型（高速 IO 侧）
- [ ] 1.1V LDO（VDD_PMCCAP）
- [ ] **VPLLVUSB 供电**（新电源轨）
- [ ] VBAT 供电（低静态电流 LDO）
- [ ] VANA 供电滤波
- [ ] VIO_Bxx 分组供电正确
- [ ] CAN 隔离侧电源（每组 DC-DC）
- [ ] RS485 隔离侧电源（每组 DC-DC）

### 1.3 千兆以太网

- [ ] PHY 选型完成（RGMII 接口）
- [ ] RGMII 信号等长布线
- [ ] RGMII 阻抗控制（50Ω 单端）
- [ ] 网络隔离变压器带磁性 RJ45
- [ ] ESD 防护器件
- [ ] MDIO 管理接口连接

### 1.4 CAN-FD 接口 (8路原生 ⭐)

- [ ] **MCAN0~MCAN7 引脚分配正确**（全部原生，无需外扩）
- [ ] CAN 收发器选型（建议带隔离）
- [ ] 避免 CAN 引脚与 JTAG/TSW/ETH 功能冲突（PINMUX 规划）
- [ ] 共模电感每路
- [ ] TVS 二极管每对
- [ ] 120Ω 终端电阻+拨码开关每路
- [ ] 气体放电管每路
- [ ] 数字隔离器每路
- [ ] 隔离电源每路
- [ ] 8 路通道布局一致性

### 1.5 RS485 接口 (8路)

- [ ] UART1~UART8 引脚分配正确
- [ ] RS485 收发器选型（半双工，带隔离）
- [ ] DE/RE 方向控制 GPIO 分配
- [ ] A 线上拉电阻、B 线下拉电阻
- [ ] 120Ω 终端电阻+拨码开关
- [ ] TVS 二极管每对
- [ ] 数字隔离器每路
- [ ] 隔离电源每路

### 1.6 UART 串口 (4路)

- [ ] UART0 (Debug) 独立引出（注意与 CAN0 复用）
- [ ] UART1~UART3 业务串口连接器定义

### 1.7 HLK-7628N 接口

- [ ] UART 接口连接正确
- [ ] 可选 TSW 交换机端口连接
- [ ] 独立供电

### 1.8 USB2.0 上位机接口

- [ ] USB Device 模式配置
- [ ] 差分阻抗 90Ω
- [ ] 差分对等长布线
- [ ] ESD 防护二极管
- [ ] VBUS 检测
- [ ] 连接器选型

### 1.9 布局布线

- [ ] 6 层板叠层确认
- [ ] USB 差分对优先布线
- [ ] RGMII 走线短且等长
- [ ] 晶振靠近主控，短走线+包地
- [ ] 强干扰接口与核心区物理隔离
- [ ] 数字地与保护地单点连接
- [ ] 电源走线载流足够（3.3V ≥2A）

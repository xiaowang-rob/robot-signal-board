# 机器人信号板 — PCB 布局布线指南

> 基于 HPM6E80IVM1 的多总线通信汇聚板
> 6 层板，BGA289(0.8mm pitch)，0405 封装

---

## 1. 叠层设计 (6 层板)

### 推荐叠层方案

| 层 | 名称 | 类型 | 说明 |
|----|------|------|------|
| L1 | TOP | 信号 + 元件 | 主元件面，HPM6E80/PHY/连接器 |
| L2 | GND1 | 地平面 | **完整地平面**，关键参考层 |
| L3 | SIG1 | 信号 + 电源 | 水平走线，部分电源走线 |
| L4 | SIG2 | 信号 + 电源 | 垂直走线，部分电源走线 |
| L5 | GND2 | 地平面 | 完整地平面 |
| L6 | BOTTOM | 信号 + 元件 | 次要元件面 |

**备选方案 (信号更多时)**:

| 层 | 名称 | 类型 |
|----|------|------|
| L1 | TOP | 信号 + 元件 |
| L2 | GND | 地平面 |
| L3 | SIG/PWR | 信号 + 电源 |
| L4 | GND | 地平面 |
| L5 | SIG/PWR | 信号 + 电源 |
| L6 | BOTTOM | 信号 + 元件 |

### 层厚建议

```
    铜厚: 1oz (外部层) / 0.5oz (内部层)
    介质: FR-4, Tg≥150°C, εr≈4.2~4.5
    板厚: 1.6mm (标准) 或 1.2mm (薄板)

    L1(1oz) ──────── 信号
          prepreg 0.1mm
    L2(0.5oz) ────── GND
          core 0.2mm
    L3(0.5oz) ────── 信号
          prepreg 0.5mm
    L4(0.5oz) ────── 信号
          core 0.2mm
    L5(0.5oz) ────── GND
          prepreg 0.1mm
    L6(1oz) ──────── 信号
```

## 2. 阻抗控制

### 目标阻抗

| 信号 | 目标阻抗 | 参考层 | 说明 |
|------|----------|--------|------|
| USB2.0 D+/D- | **90Ω ±10%** 差分 | GND | 差分对 |
| RGMII (所有信号) | **50Ω ±10%** 单端 | GND | 单端走线 |
| XPI Flash | 50Ω ±15% 单端 | GND | 如走线短可放宽 |
| CAN/RS485 | 不控阻抗 | - | 低速信号 |
| 晶振 | 50Ω | GND | 尽量短 |

### 微带线计算 (1oz 外层, εr=4.3)

| 阻抗 | 线宽 | 间距(差分) | 参考层距 |
|------|------|-----------|----------|
| 50Ω 单端 | ~8mil | - | 4mil |
| 90Ω 差分 | ~5mil | ~8mil (edge) | 4mil |

### 带状线计算 (0.5oz 内层)

| 阻抗 | 线宽 | 间距(差分) | 参考层距 |
|------|------|-----------|----------|
| 50Ω 单端 | ~5mil | - | 4mil |

> **建议**: 用 PCB 厂商提供的阻抗计算工具(如 Polar Si9000) 根据实际叠层精确计算

## 3. HPM6E80IVM1 (BGA289) 布局

### 3.1 BGA 扇出

- **封装**: 289-ball, 17×17 array, 0.8mm pitch, 14×14mm
- **扇出策略**:
  - 外层 2 圈 (A~C, P~R 行): 可在外层直接走线扇出
  - 中间区域: 需过孔到内层扇出
- **过孔**: 推荐盘中孔 (Via-in-Pad) 或 0.25mm/0.15mm (外径/内径) 通孔
- **扇出规则**:
  ```
  ┌──────────────────────────┐
  │  A B C D E F G H J K L M N P R T U  │
  │  1 ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ●   │  外层: 直接在TOP层走线
  │  ...                                │
  │  5 ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ●   │  中间: 过孔到L3走线
  │  ...                                │
  │  17 ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ●  │  底层: 直接走线
  └──────────────────────────┘
  ```

### 3.2 去耦电容布局

**原则**: 每个电源引脚近距离放置 0.1μF MLCC

```
    推荐布局 (BGA 背面贴电容):
    
    背面 (Bottom)
    ┌─────────────────┐
    │                 │
    │  C1  C2  C3     │     C1 = VDD_SOC 去耦
    │     ● BGA 中心  │     C2 = VPMC 去耦
    │  C4  C5  C6     │     C3 = VIO_Bxx 去耦
    │                 │     C4 = VANA 去耦
    └─────────────────┘     C5 = VPLLVUSB 去耦
                            C6 = 其他
```

- **0.1μF MLCC**: 0603 或 0405，距离电源焊盘 ≤50mil
- **4.7μF / 10μF 大电容**: 每电源域 1~2 个，板级放置
- 每路 0.1μF 电容通过短走线 + 过孔直接到电源焊盘正下方

### 3.3 晶振布局

```
       错误布局                   正确布局
    ┌──────────────┐          ┌──────────────┐
    │              │          │              │
    │   HPM6E80    │          │   HPM6E80    │
    │              │          │              │
    │   ────────── │          │  XTAL_IN  XTAL_OUT
    │   晶振 长走线 │          │   │  │   │  │
    │              │          │   C1  Y1  C2 │
    └──────────────┘          │   │  │   │  │
                               │  GND  │  │
                               └──────────────┘
```

- 晶振距离 HPM6E80 ≤15mm
- 走线宽度 8~12mil，最短路径
- 晶振下方 L2 层**禁止**走线和铺铜
- 包地处理: 晶振外围 GND 过孔
- 并联 1MΩ 反馈电阻靠近晶振

## 4. 高速信号布线

### 4.1 USB2.0 (90Ω 差分对)

```
    HPM6E80                    USB 连接器
    ┌──────┐                   ┌────────┐
    │      │                   │        │
    │ D+───┼────────────────┼── D+      │
    │      │   \    /        │  │        │
    │ D────┼────┼──┼────────┼── D-      │
    │      │   /    \        │  │        │
    └──────┘                   └────────┘
```

**规则**:
| 项目 | 要求 |
|------|------|
| 差分阻抗 | 90Ω ±10% |
| 差分对间距 | 保持恒定，≥8mil |
| 等长 | 对内 ≤5mil |
| 总长度 | ≤3000mil |
| 换层过孔 | ≤2 个，每过孔加 GND 回流过孔 |
| 与其他信号间距 | ≥20mil (3W) |
| 下方参考层 | **完整 GND 平面**，禁止跨分割 |
| 弯曲 | 用 45° 圆弧或斜角，避免直角 |

### 4.2 RGMII 千兆以太网 (50Ω 单端)

**信号组**:
```
    发送组: TXCK, TX_CTL, TXD[0:3]
    接收组: RXCK, RX_CTL, RXD[0:3]
    管理:   MDC, MDIO
```

**规则**:
| 项目 | 要求 |
|------|------|
| 单端阻抗 | 50Ω ±10% |
| 组内等长 | **发送组** 内 ±50mil，**接收组** 内 ±50mil |
| TCLK 到 TX 数据 skew | -1200~1200ps (由 PHY 的 delay 配置补偿) |
| 最大长度 | ≤6000mil |
| 间距 | ≥15mil (3W) |
| 参考层 | 连续 GND 平面 |
| 串联匹配 | 每信号靠近 HPM6E80 侧串 0~33Ω |
| 拓扑 | 点对点，禁止分支/桩线 |

### 4.3 XPI Flash (SPI)

> **引脚位置**: XPI0 Channel A 引脚在 **PB24~PB31** (BGA 左上角 A2~A5 区域, B04 供电分组)，Flash 芯片应就近放置在 BGA 左侧。

**规则**:
| 项目 | 要求 |
|------|------|
| 阻抗 | 50Ω ±15% (走线短时可放宽) |
| 等长 | SCLK 到各 DQ ≤200mil |
| 串联匹配 | SCLK: 22Ω, DQ: 10~22Ω (靠近 HPM6E80) |
| 拓扑 | 点对点，CS 独立走线 |
| 间距 | ≥10mil |

## 5. 电源布局

### 5.1 DCDC BUCK 布局

```
    错误布局:                   正确布局:
    ┌───────────┐              ┌───────────┐
    │ VIN ── L ── VOUT │              │ VIN ── L ── VOUT │
    │  │      │      │              │  │  ↑  │      │
    │ CIN    COUT   │  ← 长回路    │ CIN L  COUT   │
    │  │             │              │  │     │      │
    │ GND           │              │ GND NC GND    │  ← 短回路
    └───────────┘              └───────────┘
```

**规则**:
- **输入电容 CIN** 紧贴 VIN/GND 引脚
- **电感 L** 靠近 SW 引脚，走线短而宽 (≥30mil)
- **输出电容 COUT** 靠近电感输出端
- 反馈 (FB) 走线远离 SW/电感噪声区
- 功率回路面积**最小化**（CIN→L→COUT→GND）

### 5.2 电源走线宽度

| 电源轨 | 最大电流 | 最小线宽(1oz) | 建议宽度 |
|--------|----------|--------------|----------|
| 5V 输入 | 3A | 60mil | **100mil** (或铜皮) |
| 3.3V | 2A | 40mil | **80mil** (或铜皮) |
| 1.8V | 0.5A | 10mil | 30mil |
| VDD_SOC | 0.4A | 8mil | 20mil |

> 铜厚 1oz 时，1mm 宽 ≈ 0.5A (温升 10°C)

## 6. 分区布局策略

### 6.1 物理分区

```
    ┌──────────────────────────────────────────┐
    │  [HLK-7628N 区域]                        │
    │  路由器模块+天线                          │
    │                                          │
    ├───────┬──────────────────────┬───────────┤
    │       │                      │           │
    │ [CAN] │    [HPM6E80 核心区]   │  [RS485]  │
    │  8路  │    MCU+晶振+Flash   │   8路     │
    │  隔离 │                      │   隔离    │
    │       │                      │           │
    ├───────┴──────────────────────┴───────────┤
    │  [电源区]  DCDC + LDO + 隔离电源         │
    ├──────────────────────────┬───────────────┤
    │  [以太网 PHY + RJ45]    │   [USB + JTAG] │
    │                          │   调试接口     │
    └──────────────────────────┴───────────────┘
```

### 6.2 分区原则

1. **核心区居中**: HPM6E80 居中放置，晶振紧邻
2. **高速接口靠边**: USB/RGMII 靠近板边连接器
3. **隔离区明确**: CAN/RS485 隔离侧与数字侧物理分离 ≥3mm
4. **电源区集中**: DCDC/LDO 集中在板的一侧
5. **强弱分离**: 强干扰 (CAN/RS485 总线侧) 与弱信号 (MCU) 隔离
6. **连接器靠板边**: 所有对外连接器布置在板边

### 6.3 地平面分割

```
    数字地 GND (主地)        接口保护地 PGND
    ┌──────────────────┐    ┌──────────────────┐
    │                  │    │                  │
    │  HPM6E80         │    │  CAN/RS485       │
    │  PHY             │    │  隔离侧           │
    │  USB             │    │  RJ45 屏蔽层      │
    │  Flash           │    │                  │
    │                  │    │                  │
    └────────┬─────────┘    └────────┬─────────┘
             │                      │
             └────── 桥接(0Ω/磁珠) ──┘
```

- 数字地和接口保护地通过 **0Ω 电阻 / 磁珠 / 单点** 连接
- 桥接点位于板的中部，避开高速信号区域
- 隔离 CAN/RS485 的数字侧接 GND，隔离侧接 PGND

## 7. 接口信号布线

### 7.1 CAN 总线布线

```
    HPM6E80 ── 数字隔离器 ── CAN 收发器 ── TVS/CM ── 连接器
     (GND)      (GND/ISO_GND)  (ISO_GND)  (PGND)   (PGND)
```

- 8 路 CAN 通道布局**保持一致**，便于调试和降低串扰差异
- CAN 收发器到连接器走线 ≤200mm
- 隔离区宽度 ≥3mm（满足隔离耐压要求）
- CAN 差分对走线间距 10mil，远离时钟/高速信号

### 7.2 RS485 总线布线

- 同 CAN 的隔离布局要求
- DE/RE 控制信号走线简洁，远离噪声源
- 偏置电阻靠近 RS485 收发器放置
- 每路通道布局一致

## 8. PCB 工艺要求

| 项目 | 规格 |
|------|------|
| 板层 | 6 层 |
| 板材 | FR-4, Tg≥150°C, 常规或低损耗 (RGMII 走线长时) |
| 板厚 | 1.6mm (±10%) |
| 铜厚 | 外层 1oz, 内层 0.5oz |
| 最小线宽/线距 | 5/5mil (推荐 6/6mil) |
| 最小过孔 | 0.25mm 外径 / 0.15mm 内径 |
| BGA 焊盘 | 0.35mm NSMD, 阻焊桥 ≥0.1mm |
| 阻焊 | 绿色亚光 (推荐) / 黑色 |
| 表面处理 | ENIG (化金) — BGA 必需 |
| 阻抗公差 | ±10% (50Ω/90Ω) |
| 翘曲度 | ≤0.75% |
| 板名/版本丝印 | 正面空白区 + 背面 |

## 9. DFM 检查清单

- [ ] BGA 焊盘 0.35mm, NSMD, 阻焊桥 ≥0.1mm
- [ ] 最小线宽/线距 5/5mil, 推荐 6/6mil
- [ ] 过孔不落在焊盘上 (BGA 内除盘中孔外)
- [ ] 0405 封装焊盘长度/宽度符合规格
- [ ] 丝印不覆盖焊盘, 字符高度 ≥1mm
- [ ] 板边预留 5mm 禁布区 (V-cut/拼板)
- [ ] MARK 点: 3 个 (对角+1), 直径 1mm, 裸铜
- [ ] 阻焊开窗: 每个焊盘都有
- [ ] 测试点: 电源/关键信号预留测试点 (≥1mm)

## 10. 散热设计

| 器件 | 散热措施 |
|------|----------|
| HPM6E80 | BGA 下方 L2 铜皮散热过孔阵列 |
| DCDC | 大铜皮散热 + 过孔到内层 |
| 千兆 PHY | 底部铜皮 + 散热过孔 |
| HLK-7628N | 模块自带散热，预留通风孔 |

- 散热过孔: 0.3mm 外径, 1.0mm 间距阵列
- 芯片结温 Tj ≤125°C (工作环境 ≤105°C)

## 四、隔离方案决策 ⚠️

**必须先确定**: CAN 和 RS485 是否每路都做隔离？

| 方案 | CAN 器件 | RS485 器件 | 隔离电源 | BOM成本 | PCB面积 |
|------|----------|-----------|----------|---------|---------|
| **A: 全隔离** | ISO1042 ×8 | ADM2483 ×8 | B0505S ×16 | 最高 | 最大 |
| **B: 非隔离** | SN65HVD230 ×8 | SP3485 ×8 | 无 | 最低 | 最小 |
| **C: 部分隔离** | 关键通道ISO1042, 其余SN65HVD230 | 同左 | 按需 | 中等 | 中等 |

> **建议**: 机器人信号板连接的传感器/驱动器通常在同一电源系统内，**方案B(非隔离)** 足够。如果 CAN/RS485 走线较长或接外部设备，选 **方案A**。

## 五、HLK-7628N 接口要点

| 参数 | 值 |
|------|----|
| 供电 | 3.3V, **≥800mA** (推荐1A+) |
| UART | Pin31(TX0)/Pin32(RX0) + Pin15(TX1)/Pin16(RX1), 115200bps |
| 复位 | Pin14(PORST_N), 低有效, **不要外部上下拉** |
| WiFi | 2.4G 802.11b/g/n 300Mbps, Pin22(ANT)需50Ω天线 |
| 以太网 | 5口百兆交换机(PORT0~4), 每口需磁性RJ45 |
| GPIO | 8+个GPIO, 3.3V, 8mA驱动 |
| 温度 | -40~+85°C |

**信号板连接方式**:
- HPM6E80 UART1(PA07_TX/PA06_RX) → HLK Pin31(TX0)/Pin32(RX0)
- PA12(GPIO) → HLK Pin14(PORST_N), 低有效复位
- HLK PORT0 可选接外部RJ45做有线联网
